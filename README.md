Build Google AMP experiences for mobile, tablet, and desktop in minutes using a simple CLI. This template provides a ready-to-use AMP starter with sensible defaults so you can focus on content and speed instead of setup. Check out the [live preview](https://tborges.github.io/google-amp-template/) to see it in action.

## About Google AMP
The [Google AMP](https://developers.google.com/amp) project is an open-source initiative to make fast, high-performing web pages across devices and platforms. The Google AMP Cache serves cached copies of valid AMP content to deliver a consistently fast user experience. [Read more](https://developers.google.com/amp/cache/overview).

## Installation
Install and run the package directly from npm:

```shell
npm install google-amp-template
npm run google-amp-template
```

## Usage Examples
### Git clone
```shell
git clone https://github.com/tborges/google-amp-template.git
cd google-amp-template
npm run google-amp-template # Generates the AMP folder
# Open amp/index.html to view your AMP template.
```

### npm install
```shell
mkdir my-first-amp
cd my-first-amp
npm init
npm install google-amp-template
cd node_modules/google-amp-template
npm run google-amp-template # Generates the AMP folder
# Open amp/index.html to view your AMP template.
```
